## 상세 내용

### 1-1: Python 웹 프레임워크 기초

Python 기초 문법과 간단한 웹 프레임워크 개념을 학습합니다. 첫 번째 솔루션 파일을 작성하여 기본적인 코딩 환경을 구성하는 것이 목표입니다.

### 1-2: 거듭제곱 계산기

Python 함수를 활용하여 거듭제곱 연산을 수행하는 계산기를 구현합니다. 결과를 리포트 형식으로 문서화하는 연습도 포함됩니다.

### 1-3: 계산기 확장 + 음성합성

계산기를 v1에서 v2로 확장하며, Google Text-to-Speech(gTTS) 라이브러리를 활용한 음성 합성 기능을 추가합니다. 문자열 파싱 기법도 다룹니다.

### 1-4: Git 디렉토리 구조 이해

Git의 내부 디렉토리 구조(.git)를 학습하고, 정렬 알고리즘과 최대/최소값 계산기를 구현합니다.

### 1-5: Git Staging과 Commit

Git의 staging area와 commit의 차이를 실습으로 이해합니다. 정렬 계산기를 통해 코드 변경 이력을 관리하는 방법을 학습합니다.

### 1-6: Git Reset vs Revert

`git reset`과 `git revert`의 차이를 실습합니다. 실수로 커밋한 내용을 되돌리는 두 가지 방법의 장단점을 비교합니다.

### 1-7: Flask 웹 앱

Python Flask 프레임워크를 사용하여 간단한 웹 애플리케이션을 구축합니다. 라우팅, 템플릿, HTTP 요청/응답의 기초를 학습합니다.

### 1-8: Git Merge vs Rebase

`git merge`와 `git rebase`의 차이를 실습합니다. 브랜치 전략과 협업 시 어떤 방법을 선택해야 하는지 판단하는 기준을 학습합니다.
