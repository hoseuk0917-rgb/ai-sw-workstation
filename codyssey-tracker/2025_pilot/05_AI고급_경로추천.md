# 선택 AI 고급: 실시간 경로 추천 시스템

> 출처: 2025년 모집공고문 (I-nkamanda는 AI 기초 트랙을 선택하여 해당 레포 없음)

---

## 과정 개요

AI 고급 트랙은 **실시간 경로 추천 시스템**을 주제로 3.5개월간 진행됩니다. 그래프 이론과 최적 알고리즘을 기반으로 실시간 교통 정보를 반영한 경로 추천 서비스를 개발합니다.

---

## 과정 목록 (공고문 기준)

| 과정 | 주제 | 예상 핵심 기술 |
|:----:|------|--------------|
| 과정1 | 개발방법론, 그래프이론 | 애자일, 그래프 자료구조, BFS/DFS |
| 과정2 | 최적 알고리즘, 최단 경로 도출 학습 | Dijkstra, A*, Bellman-Ford |
| 과정3 | 백엔드, 프론트엔드, MVP 개발 | FastAPI/Flask, React/Vue, REST API |
| 과정4 | 최적 경로 추천 알고리즘 | 가중치 그래프, 휴리스틱 알고리즘 |
| 과정5 | 경유지, 회피지 구현, UI 구현 | 다중 경유지 처리, 지도 API 연동 |
| 과정6 | 실시간 정보 반영 개발 | WebSocket, 실시간 교통 API, 동적 재계산 |

---

## 비고

I-nkamanda는 AI 기초 트랙을 선택했기 때문에 이 과정의 실제 과제 내용은 확인할 수 없습니다. 위 내용은 공고문에 기재된 커리큘럼을 기반으로 작성되었습니다. AI 고급 트랙 수강생의 레포를 찾으면 추후 업데이트할 수 있습니다.
