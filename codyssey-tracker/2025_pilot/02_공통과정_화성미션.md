# 화성 미션 시리즈 (PBL)

> 레포: [Mars_1](https://github.com/I-nkamanda/Mars_1), [Mars-1_2](https://github.com/I-nkamanda/Mars-1_2), [Mars-2](https://github.com/I-nkamanda/Mars-2)

코디세이 과정의 PBL(Problem-Based Learning)은 **화성 기지 미션**이라는 스토리라인을 따릅니다. 수강생은 화성 기지의 다양한 문제를 해결하는 역할을 맡아 Python 프로그래밍 역량을 키웁니다.

## 상세 내용

### Mars_1: 화성 기지 로그 분석

화성 기지에서 발생한 위험 로그(`danger_logs.txt`)를 분석하여 문제 원인을 파악합니다. Python 파일 I/O와 텍스트 파싱 기법을 활용하며, 분석 결과를 `log_analysis.md`로 문서화합니다.

**주요 파일**: `danger_logs.txt`, `log_analysis.md`

### Mars-1_2: 화성 기지 자재 분석

화성 기지의 자재 데이터(CSV)를 분석하여 부족한 자재를 파악하고 보급 계획을 수립합니다. CSV 파일 처리와 데이터 분석 기초를 학습합니다.

**주요 파일**: CSV 데이터, 파트별 분석 결과

### Mars-2: 화성 미션 2탄

화성 미션의 심화 과제로, 3개 하위 폴더로 구성됩니다.

**5-1: 시저 암호 해독** — 화성 기지 도어의 암호를 시저 암호 알고리즘으로 해독합니다. 암호학 기초와 문자열 처리를 학습합니다.

**5-2: 공학 계산기** — PyQt5를 활용하여 GUI 기반 공학 계산기를 구현합니다. 이벤트 처리, 위젯 배치, 사용자 인터페이스 설계를 학습합니다.

**5-4: CCTV 시스템** — 화성 기지의 CCTV 시스템을 구현합니다. 멀티미디어 데이터 처리와 영상 관련 프로그래밍을 학습합니다.
